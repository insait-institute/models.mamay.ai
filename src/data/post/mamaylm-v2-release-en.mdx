Today INSAIT is releasing **MamayLM v2.0**, the first open multimodal LLM family adapted for the Ukrainian language. MamayLM v2.0 is built on top of Google's **Gemma 3** and combines strong instruction-following in Ukrainian and English with vision-language understanding and long context.

## What's in the release

- **Model:** `INSAIT-Institute/MamayLM-Gemma-3-27B-IT-v0.1` on [HuggingFace](https://huggingface.co/INSAIT-Institute/MamayLM-Gemma-3-27B-IT-v0.1)
- **Base:** `google/gemma-3-27b-it`
- **Context:** up to 65,536 tokens (base model supports 131,072)
- **Capabilities:** text generation, instruction following, tool calling, vision (image understanding), multi-turn conversations

## How to try it

- **Chat:** [mamay.ai](https://mamay.ai) — talk to MamayLM in the browser.
- **API:** OpenAI-compatible. See the [documentation](/docs).
- **Self-host:** weights and instructions on [HuggingFace](https://huggingface.co/INSAIT-Institute/MamayLM-Gemma-3-27B-IT-v0.1).
